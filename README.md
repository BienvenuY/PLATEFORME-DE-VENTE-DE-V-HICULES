# 🚗 CarsBusiness — Plateforme de Vente de Véhicules

Application Full-Stack de vente de véhicules d'occasion.

## 🛠️ Technologies
- **Frontend** : React, Vite, Tailwind CSS
- **Backend** : Node.js, Express
- **Base de données** : MongoDB
- **Authentification** : JWT
- **Temps réel** : Socket.IO
- **Stockage images** : Cloudinary

## ✨ Fonctionnalités
- Catalogue de véhicules filtrable
- Système de favoris
- Mode clair/sombre
- Espace administrateur complet
- Chat en temps réel
